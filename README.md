### 물구나무 푸쉬업(9-step)
---
- [ ] 파이크 푸쉬업 15회 3세트

- [ ] 파이크 푸쉬업(높게) 15회 3세트

- [ ] 헤드 to 핸드스탠드(반동) 10회 3세트

- [ ] 헤드 to 핸드스탠드(무반동) 10회 3세트

- [ ] 헤드 to 핸드스탠드(푸쉬 후 코어로 2초간 버티기) 10회 3세트

- [ ] 벽 물구나무 푸쉬업(배를 벽으로) 10회 3세트

- [ ] 벽 물구나무 푸쉬업(내려올 때 벽에서 발 떼기) 5회 3세트

- [ ] 벽 물구나무 푸쉬업(올라갈 때 벽에서 발 떼기) 5회 3세트

- [ ] 자유 물구나무 푸쉬업 3회 3세트
